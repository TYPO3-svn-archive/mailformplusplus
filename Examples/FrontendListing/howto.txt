**************************************
*    Set up this form			     *
**************************************

1. Create a new page

2. Create an "Extension template" for this page and paste the TypoScript in "typoscript_setup.txt".

3. Enter an ID of a page with a mailformplusplus plugin in plugin.F3_MailformPlusPlus.settings.fe_listing.pid

4. Create a content element of type "General plugin"

5. Select "MailformPlusPlus Listing"

6. Save and view the page.

NOTE: This will only work if the submitted form data was logged into tx_mailformplusplus_log
