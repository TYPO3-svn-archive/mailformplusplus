method generateCSV [line 239]

void/string generateCSV( misc $detailId)

Function to handle the generation of a CSV file.

Before the data gets exported, the data is checked and the user gets informed about different formats of the data. Each format has to be exported in an own file. After the format selection, the user is able to select which fields to export in a selection view. This enables the user to get rid of fields like submitted or mp-step.
